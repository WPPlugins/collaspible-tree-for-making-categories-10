== Description ==
Adds a tree view combobox on the Add/Edit Category page. Makes it easier for you to choose and navigate through categories when adding new categories.
For more information and support please visit [wikiDownload](http://wikidownload.com/ "Download Anything")

== Installation ==
Just upload the plugin folder to your wordpress plugin folder and activate the plugin from the dashboard.
